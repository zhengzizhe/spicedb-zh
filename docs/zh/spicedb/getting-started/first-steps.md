## 在 Playground 中实验

您可以在 [Playground](https://play.authzed.com) 上实验并分享模式和数据片段。

## 安装 SpiceDB 工具链

您可以使用 `zed import` 轻松地将 Playground 中的模式和数据导入到真实的 SpiceDB 实例中。

SpiceDB 和 Zed 可在 Linux、macOS 和 Windows 上运行，支持 AMD64 和 ARM64 架构。

- [安装 SpiceDB 服务器二进制文件](/zh/spicedb/getting-started/install/macos)
- [安装 SpiceDB 命令行工具：Zed](/zh/spicedb/getting-started/installing-zed)

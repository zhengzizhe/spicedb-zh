# 在 Windows 上安装 SpiceDB

本文档介绍如何在运行 Windows 的系统上安装 SpiceDB。

SpiceDB 发布适用于 AMD64 和 ARM64 架构的可执行文件归档。

## 使用 Chocolatey 安装 SpiceDB

在 Windows 上开始使用 SpiceDB 最快的方式是使用 Chocolatey：

```powershell
choco install spicedb
```

## 手动安装 Windows 版 SpiceDB 二进制文件

用户可以从 [GitHub 发布页面](https://github.com/authzed/spicedb/releases/latest)下载并解压适用于 AMD64 和 ARM64 Windows 的最新可执行文件。

# Installing SpiceDB on Windows

This document outlines how to install SpiceDB for systems running Windows.

SpiceDB publishes executable archives for both AMD64 and ARM64 architectures.

## Installing SpiceDB using Chocolatey

The quickest way to get started with SpiceDB on Windows is to use Chocolatey:

```powershell
choco install spicedb
```

## Manually Installing SpiceDB Binary for Windows

Users can download and extract the latest executable for AMD64 and ARM64 Windows from the [GitHub releases page](https://github.com/authzed/spicedb/releases/latest).

# First Steps

## Learn the Concepts

To get started with SpiceDB, we recommend understanding the following foundational concepts:

- [Writing a Schema](/en/spicedb/concepts/schema)
- [Writing Relationships](/en/spicedb/concepts/relationships)
- [Querying Data](/en/spicedb/concepts/querying-data)

If you plan on running SpiceDB, these additional concepts are recommended:

- [Commands](/en/spicedb/concepts/commands)
- [Datastores](/en/spicedb/concepts/datastores)
- [Datastore Migrations](/en/spicedb/concepts/datastore-migrations)
- [Observability](/en/spicedb/ops/observability)
- [Operator](/en/spicedb/ops/operator)

Advanced, but still fundamental concepts:

- [Consistency](/en/spicedb/concepts/consistency)
- [Caveats](/en/spicedb/concepts/caveats)

## Experiment in the Playground

You can experiment with and share schema and data snippets on the [Playground](https://play.authzed.com).

## Install the SpiceDB Toolchain

You can easily import the schema and data from the playground into a real SpiceDB instance using `zed import`.

SpiceDB and Zed operate on Linux, macOS, and Windows for both AMD64 and ARM64 architectures.

- [Install the SpiceDB server binary](/en/spicedb/getting-started/install/macos)
- [Install the SpiceDB CLI tool: Zed](/en/spicedb/getting-started/installing-zed)
